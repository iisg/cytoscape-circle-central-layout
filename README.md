# CytoscapeCentralCirclesLayout

## Description
Layout for Cytoscape.js nodes putting nodes around central nodes

## Dependencies
 * Cytoscape.js >= 2.2
 
## Installation
Require package using bower:

```
bower install iisg/cytoscape-central-circles-layout --save
```

## Usage
Call the layout with
```
cy.layout(
    name: 'centralCircles'
)
```
